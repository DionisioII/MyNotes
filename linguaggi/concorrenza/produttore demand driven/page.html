La coppia prouttore-consumatore comandata da quest’ultimo può essere convenientemente reimplementata usando una funzione lazy:

fun lazy {Produce N}
N|{Produce N+1}
end

il consumatore dovrà solo prendere la coda della lista prodotta dal produttore: automaticamente, la lista si espande. Eventualmente, lo stesso meccanismo può essere implementato usando un trigger (che è semplicemente una variante sintattica):


declare
fun lazy {Produce N}
{N|Produce N+1}
end
Ns = {Produce 0} % era lo stream object
{Browse Ns}
